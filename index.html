📱 微信公众号文章转换

专业的微信公众号文章转换服务，完美解决内容访问和格式转换问题

  • 绕过微信对特定 IP 的访问限制
  • 保留原始排版和图片，生成高质量 Markdown
  • 适配各种复杂的公众号文章格式
  • 支持直接下载和在线预览两种模式

使用方法

将微信公众号文章链接中的参数部分添加到本工具的 URL 后即可。

示例转换
原微信文章
https://mp.weixin.qq.com/s/MhzcF7u_p3UHZ9qR6hptww
转换后访问
https://wx2md-worker.[:username].workers.dev/s/MhzcF7u_p3UHZ9qR6hptww

功能选项

标准格式
直接查看 Markdown 内容，适合直接复制使用
/s/{article_id}
HTML 格式 (推荐)
HTML 包装的 Markdown，支持更好的阅读体验
/html/s/{article_id}
下载文件
直接下载 .md 格式文件到本地
/s/{article_id}?download=true

🌐 通用网页转换

新功能！支持转换任何网页为 Markdown 格式，适用于新闻文章、博客、技术文档等

使用方法

使用 /md 路径并通过 url 参数提供要转换的完整网页地址：

通用网页转换示例
原网页
https://www.example.com/article
转换后访问
https://wx2md-worker.[:username].workers.dev/md?url=https%3A%2F%2Fwww.example.com%2Farticle

参数说明

url
要转换的网页完整地址（需要 URL 编码）
format=html
以 HTML 格式查看 Markdown 内容
download=true
触发 Markdown 文件下载
